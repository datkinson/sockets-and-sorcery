Sockets and Sorcery

This project is intended to be an application that runs a socket server.

As many clients as desired can connect to the server and be shown the same map.

The map is to be the game board where a traditionally pen and paper role playing game is to be played.

A DM can control the content and players can manage their own character.

Well that is the goal...

Dependancies

  • Node
  • npm

To install just run

npm install

To run the application after install

node index.js

There are some utilities also in case you want o host this on a HTPC or larger screen, some have overscan issues so there is a screen sizer utility.
